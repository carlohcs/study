# Aula 03 - Primeiro app (Hello World)

## Existem duas maneiras de criar apps com Angular 2

### CLI

- npm install -g angular-cli
  - Requer node >= 4

- Automatização de criação de projeto com o cli:

ng new PROJECT_NAME
cd PROJECT_NAME
ng serve
