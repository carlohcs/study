# Git

## Commands

### Remove files before commit
```
git reset HEAD
```

### Remove a remote branch
```
git push origin :<branchName>
```

### Stash

#### Save the current work to apply into new work
```
git stash save
```

#### Apply the current stash to current branch
```
git stash apply
```

#### Remove the current stash
```
git stash pop
```
